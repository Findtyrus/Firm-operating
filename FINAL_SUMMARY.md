## Real-World Usage Examples

### Monday Morning Workflow
1. Sign in → See overdue tasks in red banner
2. Check "My Dashboard" → See today's tasks
3. Press **Cmd+K** → Type "River City" → Jump to client
4. Start monthly close task
5. Press **/** → Search "monthly bookkeeping" → Open SOP
6. Follow checklist step-by-step
7. Upload completed P&L to Files
8. Drag task to "Complete" in Kanban

### New Client Onboarding
1. Press **Cmd+C** → Quick create client form
2. Fill in details, set to "Prospect" status
3. Assign relationship manager
4. Press **Cmd+T** → Create "Discovery Call" task
5. Link task to client, set due date
6. Press **/** → Search "onboarding" → Open onboarding SOP
7. Follow checklist for gathering docs
8. Upload documents to Files
9. Update client status to "Onboarding" → "Active"
10. Set up recurring monthly task

### Monthly Reporting
1. Open "Financials" → See MRR dashboard
2. Switch to "Invoices" tab → Send outstanding invoices
3. Switch to "MRR Breakdown" → Review client-by-client
4. Press **Cmd+K** → "firm dashboard" → See firm-wide metrics
5. Take screenshot for partner meeting
6. Create task: "Review February Performance"

### Using SOPs Daily
1. Task pops up: "QBO Setup - New Client"
2. Click task → See SOP reference link
3. Open "Client Onboarding SOP"
4. Work through each checklist item
5. Check off steps as you complete them
6. When done, close SOP and mark task complete
