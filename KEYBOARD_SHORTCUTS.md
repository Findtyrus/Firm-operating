# ⌨️ Keyboard Shortcuts - Magnolia Advisory OS

Quick reference for all keyboard shortcuts in the system.

---

## Global Shortcuts (Work from Any Page)

| Shortcut | Action | Description |
|----------|--------|-------------|
| **⌘K** or **Ctrl+K** | Open Command Palette | Quick navigation and actions |
| **⌘T** or **Ctrl+T** | New Task | Create a new task |
| **⌘C** or **Ctrl+C** | New Client | Create a new client |
| **/** | Search/Open Palette | Open command palette (when not in input field) |
| **Esc** | Close Palette | Close command palette |

---

## Command Palette Navigation

Once the palette is open (⌘K):

### Jump to Pages
- Type **"dash"** → Firm Overview Dashboard
- Type **"my"** → My Dashboard
- Type **"client"** → Clients page
- Type **"task"** → Tasks page
- Type **"sop"** → SOPs page
- Type **"fin"** → Financials page
- Type **"file"** → Files page
- Type **"set"** → Settings page

### Quick Actions
- Type **"new task"** → Create task
- Type **"new client"** → Create client
- Type **"new sop"** → Create SOP

### Navigation
- **↑/↓** - Move between options
- **Enter** - Select option
- **Esc** - Close palette

## Browser Shortcuts Still Work

Standard browser shortcuts remain functional:

- **⌘R** / **Ctrl+R** - Refresh page
- **⌘W** / **Ctrl+W** - Close tab
- **⌘T** / **Ctrl+T** - New tab (conflicts with our New Task, use in new tab)
- **⌘+** / **Ctrl++** - Zoom in
- **⌘-** / **Ctrl+-** - Zoom out

---

## Tips for Power Users

1. **Start every session with ⌘K** - Check what needs attention
2. **Use ⌘T daily** - Quickly create tasks as they come up
3. **Search SOPs with /** - When you need to reference a procedure
4. **Keep palette open** - Type, select, close - very fast workflow
5. **Learn the patterns** - After a few uses, muscle memory takes over
